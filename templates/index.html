Background

Client
- Lynne Koropp, Technical Liaison for Absolute Media Inc.

Client's Business
- Absolute Media Inc. (AMI) is a privately-owned media planning and buying company specializing in maximizing the return of our clients valuable advertising funds.

Client's Problem to be Solved
- Our Client wants a new inventory system that replaces their current one. That reimplements current protocols of their old system (tracking orders of products and issuing packing slips).

Scope

The goal of this project is to replace the outdated database with a new system that can be run on modern technology and to be able to access the system with ease.

Proposed Solution

Our proposed solution for our client is to create a new inventory tracking system from scratch. We decided on doing this rather than continuing to build off original system because the original system is extremely outdated and runs on ancient hardware and software. Windows XP in 2021 is unacceptable! Our solution can solve our clientâ€™s problem because it brings their system up to speed with modern technology and makes it more convenient. With this new application, they will be able to do their work on modern browsers and operating systems and not worry about maintaining old hardware. The new system will have an updated user interface and follow more modern design trends.

Highlights

Compared to Absolute Media Inc.'s current webpage and database system, our product is meant to be a much more clean and polished version of what they currently have. Our project is special to us because it's designed to be a much more efficient system for Absolute Media such as making processes like tracking companies, issue packing slips, and input integration much easier and clean. By comparison to what Absolute Media currently has, the new webpage/database system is much more innovative.

Our solution, while it's functionality is primarily based on their current system, shares some resemblance of other platforms such as Discord and Slack in terms of it's user interface. It shares some qualities such that it looks clean, colors contrast well, and it's easy to find things. Given the functionality of the webpage/database system and the new user interface makes our project unique to other company platforms.
